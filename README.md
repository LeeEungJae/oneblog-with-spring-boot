# oneblog-with-spring-boot
게시판만들기 with spring boot, security, rest API

### 👨‍🎓학습목표
* Self-Describtive Message와 HATEOAS를 만족하는 REST API를 이해합니다.
* 스프링 HATEOAS와 스프링 REST Docs 프로젝트를 활용할 수 있습니다.
* 테스트 주도 개발(TDD)에 익숙해 집니다.
* Error Customizing
* MSA 기반 설계 목표
* React 간단하게 적용해보기
* Java stream API

## 프로젝트 정보
* Spring boot 2.5
* JDK 11
* H2, MySQL
* React

## 의존성
* Web
* JPA
* HATEOAS
* TEST Docs
* H2
* MySQL
